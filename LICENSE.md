6. **Keep MacBook Executor updated** — The **MacBook executor update** system checks automatically, but always verify you're on the latest version shown in the Discord `#releases` channel.

## 🔧 Common MacBook Issues & Fixes

### ❌ "MacBook Executor won't open — 'unidentified developer' error"

**Fix:** This is macOS Gatekeeper protecting you — it's NOT a virus warning. 
1. Click **Cancel** on the popup
2. Go to **System Settings** → **Privacy & Security** 
3. Scroll to **Security** section
4. Click **"Open Anyway"** next to MacBook Executor
5. Click **Open** in the confirmation dialog

You only need to do this once.

---

### ❌ "Executor crashes immediately on launch"

**Fix for Apple Silicon MacBooks (M1/M2/M3/M4):**
- Ensure you downloaded the correct ARM64 version
- Delete the app and re-download from the [official website](https://macbookexecs.github.io/macbook/)

**Fix for Intel MacBooks:**
- Verify macOS version is 12 Monterey or newer
- Install Rosetta 2: Open Terminal and run `softwareupdate --install-rosetta`

---

### ❌ "Scripts execute but nothing happens in Roblox"

**Fix:** The script may be outdated or game-patched.
- Check the **Discord server** `#script-hub` channel for updated versions
- Make sure MacBook Executor shows "Attached" (green indicator) before executing
- Try a different script to isolate the issue

---

### ❌ "My MacBook gets hot / fans spin loudly"

**Fix for MacBook Pro (with fans):**
- Close other resource-heavy apps (Chrome, Zoom, Discord, etc.)
- Check Activity Monitor for CPU usage — MacBook Executor should use <15% CPU
- Ensure you're not running multiple scripts simultaneously

**Fix for MacBook Air (fanless):**
- This is normal during extended scripting sessions
- Take a 10-minute break every 2 hours to let the MacBook cool down
- Reduce graphics settings in Roblox (lower FPS = less heat)

---

### ❌ "Injection fails / 'Attach failed' message"

**Fix:** 
1. Close MacBook Executor completely (Cmd+Q)
2. Close Roblox completely
3. Re-launch Roblox and join a game
4. Wait 10 seconds for the game to fully load
5. Launch MacBook Executor and click Inject

If the problem persists, check the Discord `#help` channel.

---

### ❌ "MacBook Executor says 'Update Available' but won't update"

**Fix:** 
- Download the latest version manually from our [official website](https://macbookexecs.github.io/macbook/)
- Delete the old version from Applications folder
- Install the new version following the [installation guide](#how-to-install-on-macbook)
- Your scripts and settings will be preserved (stored separately in `~/Library/Application Support/MacBookExecutor`)
